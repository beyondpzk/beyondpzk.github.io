参考核心文献：Visual Autoregressive Modeling: Scalable Image Generation via Next-Scale Prediction (VAR)

导言

以GPT-4为代表的大语言模型（LLM）在人工智能领域引发的范式革命。LLM的核心逻辑极其简单——基于自回归（Autoregressive, AR）的“下一个Token预测（Next-token prediction）”。然而，这种简单的逻辑却衍生出了两种令人惊叹的属性：Scaling Laws（缩放定律）和Zero-shot Generalization（零样本泛化能力）。

在计算机视觉领域，我们同样渴望构建出具备此类强大泛化与扩展能力的“世界模型（World Models）”。然而，传统的视觉自回归模型（如VQGAN、Parti等）虽然模仿了LLM的自回归机制，但在图像生成质量上长期落后于扩散模型（Diffusion Models，如Sora、Stable Diffusion 3背后的DiT架构），且计算复杂度极高。

我们深入研讨一篇可能改变这一格局的重量级工作：VAR (Visual AutoRegressive Modeling)。这项工作首次让GPT风格的自回归模型在图像生成质量上全面超越了强力的扩散模型（DiT），并在推理速度、数据效率和可扩展性上展现出压倒性优势。本节课我们将从理论基础、数学推导、模型架构、算法实现到实验分析，对VAR进行全方位的深度解剖。

第一部分：传统视觉自回归模型的理论困境与数学反思

理解传统视觉自回归模型究竟“错”在哪里。

1.1 传统“Next-Token Prediction”在图像上的生搬硬套

要将图像输入自回归模型，传统方法分为两步：

  1. Token化（Tokenization）：通过VQVAE（向量量化变分自编码器）将连续的2D图像映射为离散的Token网格 $q \in [V]^{h \times w}$，其中 $V$ 是词表大小。
  2. 展平与排序（Flattening & Ordering）：将2D网格展平为1D序列 $x = (x_1, x_2, \dots, x_{h \times w})$，通常采用光栅扫描顺序（Raster-scan，即从左到右、从上到下）。

在此基础上，模型去最大化序列的对数似然： \(p(x_1, x_2, \dots, x_T) = \prod_{t=1}^T p(x_t \mid x_1, x_2, \dots, x_{t-1})\)

1.2 传统范式的四大致命缺陷

传统范式看似合理，但在数学与物理直觉上存在显著缺陷：

  1. 违背了自回归的单向依赖数学前提：VQVAE的编码器通常是深层CNN或Transformer，其输出的特征图 $f$ 上的每个向量 $f^{(i,j)}$ 是全局相互依赖的。但是，自回归强制要求 $x_t$ 仅依赖于其前缀 $x_{<t}$。这种在特征提取时双向耦合，却在生成时强行切断单向依赖的做法，导致了严重的理论裂痕。
  2. 破坏了图像的空间结构局部性（Structural Degradation）：在2D网格中，Token $q^{(i,j)}$ 与其上下左右的Token高度相关。一旦展平成1D序列，原本相邻的上下方Token在序列中被强行拉远了 $w$ 个时间步。
  3. 零样本泛化能力受限：纯粹的单向扫描导致模型无法实现双向推理。例如，给定图像的下半部分去推断上半部分，或者给定外围去推断中心（Out-painting/In-painting），光栅扫描的自回归模型难以直接做到。
  4. 计算复杂度极高（灾难性的 $O(n^6)$ 复杂度）： 我们来推导一下传统AR的复杂度。设图像量化后的宽和高为 $n$，则总Token数为 $n^2$。 在第 $i$ 步（$1 \le i \le n^2$）自回归解码时，新Token需要与前面 $i$ 个Token进行注意力计算，计算量为 $O(i^2)$（注：不使用KV Cache的情况下全量计算，或使用KV Cache但考虑序列总和）。 计算整个序列的总复杂度为：$\sum_{i=1}^{n^2} i^2 \approx \frac{1}{3}(n^2)^3 = O(n^6)$。 这意味着，当分辨率翻倍时，计算量会以64倍的速度爆炸，导致传统AR极难扩展到高分辨率生成。

第二部分：VAR的核心理论——“下一个尺度预测”

人类作画或认知世界，绝非是从左上角到右下角逐像素扫描，而是从全局结构到局部细节（Coarse-to-fine）的层次化递进。VAR的精髓即在于此。

2.1 数学表征：多尺度Token映射

VAR不生成一维序列，而是生成一系列尺度递增的2D Token图（Token Maps）： $R = (r_1, r_2, \dots, r_K)$，其中 $r_k$ 是一张尺寸为 $h_k \times w_k$ 的Token分布图。 尺度的定义满足 $h_1 < h_2 < \dots < h_K$，最终尺度等于原始图像量化后的分辨率。

VAR对图像的联合似然分布进行了重新分解： \(p(R) = \prod_{k=1}^K p(r_k \mid r_1, r_2, \dots, r_{k-1})\)

在这个公式中，$r_k$ 内部的所有Token是条件独立且并行生成的，它们仅仅依赖于先前生成的低分辨率尺度图 $r_{<k}$。

2.2 VAR的时间复杂度优势证明：$O(n^4)$

我们来严谨推导VAR的复杂度。 假设最终尺度为 $n \times n$（即 $h_K = w_K = n$），我们设定相邻尺度的分辨率按比例 $a > 1$ 递增，即 $n_k = a^{k-1}$，并满足 $a^{K-1} = n$。 在第 $k$ 个尺度进行自回归生成时，前缀中包含的总Token数为前 $k$ 个尺度的Token数之和： \(\text{Tokens}_{\le k} = \sum_{i=1}^k n_i^2 = \sum_{i=1}^k a^{2(i-1)} = \frac{a^{2k}-1}{a^2-1}\) 第 $k$ 步生成时的注意力计算复杂度大约是序列长度的平方，即： \(\left( \frac{a^{2k}-1}{a^2-1} \right)^2\) 对所有 $K$ 个尺度求和，并结合 $K = \log_a(n) + 1$，利用等比数列求和公式可以严格证明，总体计算复杂度收敛于 $O(n^4)$。 从 $O(n^6)$ 降维到 $O(n^4)$，这是VAR能在推理速度上比传统VQGAN快20倍的核心数学基础。

第三部分：模型架构深度解析

VAR的实现需要协同设计两个核心组件：支持多尺度的量化自编码器（Multi-scale VQVAE）和预测下一尺度的Transformer（VAR Transformer）。我们需要对这两部分的结构进行细致拆解。

3.1 Multi-scale VQVAE的架构与算法

VAR之所以能够进行“下一个尺度”的预测，前提是我们必须有一个特殊的Tokenizer，能够将图像编码为多尺度的残差Token。

结构设计要点：

  • 基础架构：沿用标准的VQVAE（CNN构成的Encoder $E$ 和 Decoder $D$）。
  • 统一词表：所有尺度共享同一个大小为 $V$ 的Codebook $Z \in \mathbb{R}^{V \times C}$，这保证了自回归Transformer输入/输出词表的一致性。
  • 残差量化机制：从低分辨率到高分辨率，每一层量化网络只学习去弥补前一层丢失的细节。

编码算法深度剖析（Algorithm 1）：

  1. 给定原始图像 $im$，通过Encoder提取最高分辨率的连续特征图 $f = E(im)$。
  2. 循环遍历 $K$ 个尺度（从 $k=1$ 到 $K$）：
    • 对特征图 $f$ 进行空间插值降采样到当前分辨率 $h_k \times w_k$。
    • 对降采样后的特征进行最近邻量化 $Q(\cdot)$，得到当前尺度的Token离散索引图 $r_k$。
    • 从Codebook $Z$ 中查表得到连续表征 $z_k = \text{lookup}(Z, r_k)$。
    • 关键残差步骤：将 $z_k$ 插值放大回原始最高分辨率 $h_K \times w_K$。
    • 使用一个尺度专属的卷积层 $\phi_k$ 对放大的 $z_k$ 进行处理，并从原特征图 $f$ 中减去它：$f \leftarrow f - \phi_k(z_k)$。
  3. 最终得到一个包含多尺度Token索引的列表 $R = [r_1, r_2, \dots, r_K]$。

思考题留白：为什么要在特征空间 $f$ 进行逐尺度残差相减？ 因为这样强制要求较高分辨率的Token（如 $r_{k+1}$）只去编码低分辨率（$r_k$）无法表达的高频残差细节。这极大地减少了尺度间的信息冗余。

解码算法（Algorithm 2）： 解码过程则是编码的逆向累加。

  1. 初始化空的高分辨率特征图 $\hat{f} = 0$。
  2. 循环提取 Token 图 $r_k$，查表得到 $z_k$。
  3. 将 $z_k$ 插值放大到最高分辨率，经过卷积 $\phi_k$ 后累加到 $\hat{f}$ 上：$\hat{f} \leftarrow \hat{f} + \phi_k(z_k)$。
  4. 累加完所有尺度后，将最终的 $\hat{f}$ 送入 Decoder 重构出图像 $\hat{im} = D(\hat{f})$。

3.2 VAR Transformer 的微观架构

有了多尺度的 Token，我们就可以训练 GPT 风格的 Transformer 了。

网络结构： VAR 采用了无显式位置编码的纯 Decoder-only Transformer 架构，与 GPT-2 极其相似，保持了最大的算法通用性。 为了引入类别条件（Class Conditional），模型使用了 自适应层归一化（Adaptive Layer Normalization, AdaLN）。我们将类别嵌入（Class Embedding）不仅作为序列的起始符 [s] 喂入网络，同时也将其输入到 MLP 网络中，生成缩放和平移参数（$\gamma, \beta$），去调制Transformer每个Block内部的特征分布。

参数设定（Scaling Rules）： 为了深入研究缩放定律（Scaling Laws），论文严格遵循了一种线性缩放规则来设计模型族群。给定模型的深度（层数）$d$：

  • 模型宽度（隐含层维度）：$w = 64 \times d$
  • 注意力头数（Attention Heads）：$h = d$
  • 此时，模型的参数量 $N$ 与深度 $d$ 的三次方成正比：$N \approx 73728 \times d^3$。这为后续严密的Scaling Law分析提供了架构基础。

前向传播与注意力掩码（Attention Mask）机制： 序列输入结构是将 $K$ 个尺度的 Token 图展平后依次拼接：[s], r_1, r_2, ..., r_{K-1} 去预测 r_1, r_2, ..., r_K。 这里的 Attention Mask 是 VAR 的灵魂：

  • $r_k$ 尺度中的任意一个 Token，可以看见（Attent to）所有的起始符 [s] 以及所有之前尺度 $r_{<k}$ 的 Token。
  • 但是，$r_k$ 尺度内部的 Token 之间不可见（它们是条件独立的），也不能看见后续尺度 $r_{>k}$ 的 Token。
  • 这种 Block-causal 的掩码机制，使得 $r_k$ 阶段的所有 Token 可以在一次前向传播中完全并行生成，极大地提升了推理速度。

第四部分：实验结果、扩展定律与涌现能力

这部分我们通过数据和现象，来印证VAR范式为什么被称为“破局者”。

4.1 压倒性的生成性能

在 ImageNet 256x256 和 512x512 的类条件生成基准测试中，VAR 以 2B（20亿）参数量的模型，达成了革命性的指标：

  • 图像质量：FID 降至 1.73，Inception Score (IS) 飙升至 350.2。作为对比，曾经震撼业界的 DiT-XL/2 (675M) FID为2.27，甚至最新的 L-DiT-7B FID也仅为 2.28。
  • 推理效率：由于 $O(n^4)$ 复杂度与并行解码机制，VAR 只需要 10 步就可以生成一幅高斯质量的图像，而传统 VQGAN 需要 256 步，DiT 更是动辄需要 250 步的扩散去噪阶段。VAR 在推理阶段比 DiT-XL/2 快出 45 倍以上。

4.2 视觉领域的 Power-Law Scaling Laws（核心学术亮点）

LLM 最伟大的特性在于“模型越大，效果必然越好”，这被称为幂律缩放定律。在视觉领域，由于传统生成的诸多限制，很少观察到如此清晰的曲线。VAR 团队训练了从 18M 到 2B 的 12 个不同规模的模型，发现了惊人的规律：

  1. 随着参数量 $N$ 的 Scaling： 测试集交叉熵损失 $L$ 以及 Token 预测错误率 $\text{Err}$ 均与参数量呈现完美的幂律下降（Pearson 相关系数高达 -0.998）： \(L_{\text{avg}} = (2.5 \times N)^{-0.20}\)
  2. 随着最优计算量 $C_{\min}$ 的 Scaling： 当绘制不同模型在训练计算量（以 PFlops 计）上的 Pareto 前沿时，同样呈现出完美的幂律定律： \(L_{\text{avg}} = (1.5 \times 10^{-5} C_{\min})^{-0.16}\) 启示：这一结果具有里程碑意义。它表明 VAR 框架具备类似 LLM 那样极高的模型容量预测性。只要堆算力、加参数，视觉生成的质量就会稳定上升，没有天花板（直至逼近香农信息论下限）。

4.3 零样本泛化能力（Zero-shot Generalization）

VAR 解决了传统 AR 被破坏的空间局部性，因此展现出了令人瞩目的下游任务零样本迁移能力（不需要对任务进行微调）：

  • Image In-painting（图像补全）与 Out-painting（外推扩图）：利用模型生成时天然维持的空间二维结构，我们只需要将保留区域在各个尺度的 Token 强制注入，模型就能通过 Attention 自适应地并行生成缺失区域的 Token。
  • Image Editing（图像编辑）：由于类别标签控制 AdaLN，可以通过改变类别标签并限制某些低分辨率尺度的不变性，实现不改变原图大体结构的语义修改。
